# Food_care_frontend_project
Project Overview

Food Care is a responsive frontend web project developed to spread awareness about food waste management and food donation. The website provides a simple and user-friendly interface where users can explore donation-related sections and navigate through different pages easily.

Features-
Responsive homepage design
User Registration page
Login page
Food Donation section
Easy navigation between pages
Clean and user-friendly interface


Technologies Used
HTML
CSS
JavaScript
Project Purpose

The main purpose of this project is to promote food donation and reduce food waste by creating an interactive web platform.
